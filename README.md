## 1. Overview

This module can control on/off and read current whether AC or DC cureent. The current data from ACS712-05A will be receive to ADC which is on the BLE. 


<a name="Pinouts"></a>
## 2. Pinouts

* AO
	Current data
* DIN
	Contorl on/off
* 5VCal
	Reference Voltage
* PIR
	Read PIR state
